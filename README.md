# ♟️ Two Player Chess (Keyboard + Mouse)

A fully functional **two-player chess game** built using **HTML, CSS, and JavaScript**, supporting both **keyboard navigation** and **mouse interaction**.

---

## 🌐 Live Demo

👉 https://vivekroy8584-sketch.github.io/Chess/

---

## 🚀 Features

- ♜ Complete chess rules implementation  
- 🎮 Play locally (2 players on same device)  
- ⌨️ Keyboard controls + 🖱️ mouse support  
- ♔ Legal move validation (no illegal moves allowed)  
- ♚ Check, Checkmate & Stalemate detection  
- ♟️ Pawn promotion with selection UI  
- 🔁 Undo move functionality  
- 🆕 New game reset option  
- 🎯 Highlighting:
  - Selected piece  
  - Legal moves  
  - Last move  
  - King in check  

---

## 🖥️ Controls

### 🎮 Keyboard

| Key | Action |
|-----|--------|
| ↑ ↓ ← → | Move cursor |
| Enter / Space | Select / Move piece |
| Esc | Deselect |
| U | Undo move |
| N | New game |

---

### 🖱️ Mouse

- Click a piece → Select  
- Click a valid square → Move  

---

## 🧠 Game Logic

This project implements:

- Piece movement rules (Pawn, Rook, Knight, Bishop, Queen, King)  
- Castling (King-side & Queen-side)  
- En passant  
- Pawn promotion  
- Check detection  
- Checkmate & stalemate evaluation  
- Move history tracking
